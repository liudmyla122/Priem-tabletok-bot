# Бесплатные варианты хостинга для Med Reminder Bot

## 1. Render (⭐ Рекомендуемый — самый простой!)
- **Ссылка: https://render.com/
- **Что даёт:
  - Бесплатный Node.js веб-сервис
  - Бесплатный фоновый воркер
  - Бесплатный PostgreSQL (90 дней)
  - Бесплатный Redis
- **Ограничения:
  - Веб-сервисы "спят" после 15 минут неактивности (не критично для бота)
  - Воркер: 750 часов/месяц (достаточно)
- **Как настроить: см. файл `RENDER_DEPLOY.md`

## 2. AWS Free Tier (у вас уже есть аккаунт!)
- **Ссылка: https://aws.amazon.com/free/
- **Что даёт:
  - Бесплатный EC2 инстанс t2.micro/t3.micro в течение 12 месяцев
  - Бесплатный RDS PostgreSQL в течение 12 месяцев
  - Бесплатный ElastiCache Redis в течение 12 месяцев
- **Ограничения:
  - Действует только 12 месяцев
  - Нужно следить, чтобы не выйти за лимиты
- **Как настроить: см. файл `AWS_DEPLOY.md`

## 3. Railway
- **Ссылка: https://railway.app/
- **Что даёт:
  - Бесплатный кредит $5 в месяц
  - Поддерживает Node.js, PostgreSQL, Redis
- **Ограничения:
  - Лимит $5 в месяц (для небольшого бота это достаточно)
- **Как настроить:
  1. Зарегистрируйтесь на Railway
  2. Создайте новый проект и подключите репозиторий
  3. Добавьте сервисы PostgreSQL и Redis
  4. Добавьте переменные окружения (в том числе `DATABASE_URL` и `REDIS_URL`)
  5. Добавьте два сервиса: один для бота (`npm run start`), другой для воркера (`node dist/scheduler/worker.js`)

## 4. Oracle Cloud Free Tier (бесплатно НАВСЕГДА!)
- **Ссылка: https://www.oracle.com/cloud/free/
- **Что даёт:
  - 2 бесплатных VPS (Ampere A1 Compute, 4 CPU и 24 ГБ памяти всего)
  - 2 бесплатных Базы данных (Oracle Autonomous Database)
- **Ограничения:
  - Нужно указать кредитную карту (но деньги не снимаются, если вы не выходите за лимиты)
  - Интерфейс немного сложнее, чем у Render
- **Как настроить:
  1. Зарегистрируйтесь на Oracle Cloud Free Tier
  2. Создайте Compute инстанс (VM)
  3. Подключитесь по SSH и установите всё необходимое, как в инструкции для AWS EC2

## 5. Fly.io
- **Ссылка: https://fly.io/
- **Что даёт:
  - Бесплатный кредит $5 в месяц
  - Поддерживает Node.js, PostgreSQL, Redis
- **Ограничения:
  - Лимит $5 в месяц

---

**Рекомендация: Начните с **Render** — это самый простой и быстрый способ запустить бот бесплатно!
