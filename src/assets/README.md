# Анимации напоминаний

Положи сюда готовые видео-анимации (формат MP4, короткие, 2-5 секунд, вертикальные или квадратные)
со следующими именами — код их подхватит автоматически:

- `pill-morning.mp4` — напоминание про утренний приём лекарства
- `pill-day.mp4` — дневной приём
- `pill-evening.mp4` — вечерний приём
- `appointment-reminder.mp4` — напоминание про визит к врачу
- `escalation-alert.mp4` — более тревожная анимация для повторного напоминания

## Как быстро сделать такие анимации без дизайнера

1. **Canva** (бесплатно) → шаблоны "Instagram Story" → анимированные стикеры таблеток/календаря → экспорт в MP4
2. **LottieFiles.com** → бесплатные Lottie-анимации на тему "medicine reminder", "pill", "calendar" → конвертировать в MP4 через встроенный экспорт
3. Через **ffmpeg**, если у тебя уже есть статичная картинка + хочешь сделать лёгкую анимацию (zoom/pulse):
   ```bash
   ffmpeg -loop 1 -i pill.png -vf "zoompan=z='min(zoom+0.0015,1.2)':d=100" -t 3 -c:v libx264 -pix_fmt yuv420p pill-morning.mp4
   ```

Если файлов здесь нет — бот всё равно полностью работает, просто отправляет обычное текстовое
сообщение с кнопкой вместо анимации (это заложено в коде как автоматический fallback).
