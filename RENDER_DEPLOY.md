# Деплой на Render (БЕСПЛАТНО!)

Render — это самый простой бесплатный сервис для вашего бота! Он поддерживает Node.js, PostgreSQL и Redis — всё что нужно для нашего проекта.

## Предварительные шаги

1. Зарегистрируйтесь на [Render](https://render.com/) (можно через GitHub)
2. Загрузите ваш проект на GitHub/GitLab
3. Убедитесь, что в корне проекта есть файл `render.yaml` (он есть, я его создал)

## Шаги деплоя:

1. Перейдите в [Render Dashboard](https://dashboard.render.com/)
2. Нажмите **"New +" → "Blueprint"**
3. Подключите ваш GitHub/GitLab репозиторий с ботом
4. Выберите репозиторий и ветку (например, `main` или `master`)
5. Нажмите **"Apply"**
6. Render автоматически создаст все сервисы:
   - Базу данных PostgreSQL (PostgreSQL)
   - Redis
   - Веб-сервис для бота
   - Воркер для отправки напоминаний
7. После создания всех сервисов перейдите в каждый сервис и добавьте **Environment Variables (переменные окружения):

   **Обязательно нужно добавить:
   - `BOT_TOKEN`: ваш токен от @BotFather

   **Опционально (если используете):
   - `TWILIO_ACCOUNT_SID`
   - `TWILIO_AUTH_TOKEN`
   - `TWILIO_PHONE_NUMBER`
   - `TELEGRAM_LIQPAY_PROVIDER_TOKEN`
   - `OPENAI_API_KEY`
   - `TTS_PROVIDER`
   - `TTS_VOICE`
   - `PUBLIC_URL: это будет URL вашего веб-сервиса на Render (например, `https://medbot-server.onrender.com`)

8. Чтобы добавить переменные:
   - Перейдите в сервис `medbot-server`
   - Нажмите **Environment**
   - Добавьте нужные переменные и нажмите **Save Changes**
   - Сделайте то же самое для сервиса `medbot-worker`

9. Дождитесь, пока сервисы пересоберутся и запустятся.

Всё! Ваш бот теперь работает бесплатно 24/7 на Render!

## Ограничения бесплатного плана Render:
- Веб-сервисы "спят" после 15 минут неактивности (но это не страшно для бота, потому что бот работает через long polling)
- Воркерам есть лимит 750 часов в месяц (это достаточно для одного воркера)
- База данных PostgreSQL бесплатна 90 дней, затем нужно перейти на платный план
